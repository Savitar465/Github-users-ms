# 🚀 CÓMO EMPEZAR - Guía de entrada rápida

Bienvenido a **ms-usuarios**. Elige tu rol:

## 👤 Por rol

### 🆕 Dev nuevo (30-70 min)
1. Leer QUICK_START.md (10 min)
2. Ejecutar localmente (10 min)
3. Leer README-STARTER.md (15 min)
4. Revisar ARCHITECTURE.md § 10 CRUD (30 min)
→ Listo para PR

### 💻 Implementar feature
1. Leer ARCHITECTURE.md § 10 (CRUD canónico)
2. Leer README-STARTER.md § 2-3 (convenciones)
3. Usar como plantilla
4. PR con template automática
→ Feature con calidad

### 👀 Code reviewer
1. Leer README-STARTER.md § 5 (Do/Don't)
2. Revisar contra .github/pull_request_template.md
3. Validar ARCHITECTURE.md (capas)
4. Feedback accionable
→ Reviews consistentes

### 🐛 Reportar issue
1. GitHub → Issues → New
2. Template automática (no dejar en blanco)
3. Rellenar campos requeridos
→ Issue claro

### 🎯 Navegar documentación
Leer DOCUMENTATION.md (matriz de decisión)
→ Orientación clara
